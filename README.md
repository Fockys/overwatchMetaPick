Overwatch HERO info website

This small project was created as a way to improve my web development skills, first time using nextjs.

Website for providing information about different heros from the game overwatch.
Has a login page and dashboard for an admin account to alter hero informatiomation.

Information about the heros is stored on a backend MYSQL database.
Uses next.js framework with drizzle ORM, tailwindCSS and ESLINT

Adming pages, dashboard and admin api are protected with authentication.
Authentication is cookie based

Includes error handling for when services such as the database are down.


Designed to be used with a MySQL database
Data is properly relational and passwords are hashed
