Overwatch HERO info website

Website for hosting information about different heros from the game overwatch.
Has a login page and dashboard for an admin account to alter hero informatiomation.

Information about the heros is stored on a backend MYSQL database.


This small project was created as a way to improve my web development skills, first time using nextjs.

This project uses
NextJS
Drizzle
TailwindCSS
ESLINT


Designed to be used with a MySQL database
